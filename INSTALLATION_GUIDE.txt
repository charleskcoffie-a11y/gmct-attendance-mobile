� HOW TO CLOSE THE APP IN FULL-SCREEN MODE

Since the app runs in full-screen mode like a native app, there's no close button.
Use your phone's standard gestures to close it:

🍎 IPHONE:
  • Swipe up from the very bottom of the screen
  • Pause briefly in the middle to see all open apps
  • Swipe up on GMCT Attendance app to close it
  
  Quick Exit to Home Screen:
  • Just swipe up quickly from the bottom → goes to home screen
  • App stays running in background (normal iPhone behavior)

🤖 ANDROID:
  • Swipe up from the bottom and hold (or tap Recent Apps button)
  • You'll see all your open apps
  • Swipe the GMCT Attendance app away to close it
  
  Quick Exit to Home Screen:
  • Swipe up quickly from bottom (or tap Home button)
  • App stays running in background (normal Android behavior)

💡 NOTE: This is how ALL full-screen apps work on mobile phones!

ℹ️ HOW THE APP WORKS

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FOR CLASS LEADERS:

Login:
  • Enter your member number/ID as username
  • Enter your password (first login default: gmct2026)
  • You'll see your class number and options

Mark Attendance:
  • Tap "Mark" to record attendance for today
  • Select date, select service (Sunday or Bible Study)
  • Mark each member: Present ✓ / Absent ✗ / Sick 🤒 / Travel ✈️
  • Tap "Save" to submit
  • Data saves locally even if you're offline

View Records:
  • Tap "Records" to see all past attendance records
  • Organized by year and month
  • Click any record to expand details
  • Click edit button (pencil) to update an old record

View Reports & Stats:
  • Tap "Reports" to see attendance statistics
  • See monthly/quarterly reports
  • Track member trends and absence patterns

Profile Settings:
  • Tap "Profile" to update personal info
  • Change password if needed

🆘 TROUBLESHOOTING

Problem: App won't install
  → Make sure you're using Safari (iPhone) or Chrome (Android)
  → Check you have enough storage space

Problem: Can't log in
  → Make sure username is your member number/ID (e.g., 123)
  → First login default password is gmct2026
  → Check internet connection
  → Clear browser cache if needed

Problem: Data not saving
  → Check "Sync" tab - may be syncing
  → Ensure you tapped "Save" button
  → Try marking attendance again

Problem: Slow loading
  → Close other browser tabs
  → Clear browser cache
  → Check internet connection
  → Try reloading the app

Problem: Records not showing
  → Check that you're in the correct class
  → Make sure records were actually saved
  → Refresh by going to another tab and back
